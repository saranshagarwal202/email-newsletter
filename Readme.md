# Email Newsletter Service

A high-throughput, low-latency email newsletter backend service built in Rust.

## About

This project is a learning exercise in building production-grade REST APIs using Rust. It serves as a microservice backend for managing email newsletter subscriptions and deliveries, suitable for blogs, content platforms, or any application requiring newsletter functionality.

## Learning Source

Following the [Zero To Production in Rust](https://www.zero2prod.com/) book by Luca Palmieri.

## Tech Stack

- **actix-web**: High-performance web framework
- **serde**: Serialization/deserialization
- **tokio**: Async runtime

## Getting Started

```bash
# Run the server
cargo run

# Run tests
cargo test
```
